# Changelog

All notable changes to the HyperSentry Labs organization-wide `.github`
repository are documented in this file. Individual projects maintain their
own changelogs; this file covers only shared templates, workflows, and
policies.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html)
for its own conventions.

## [Unreleased]

### Added

- Initial organization profile README
- Community health files: Code of Conduct, Contributing, Security, Support,
  Governance, Maintainers, Community
- Issue forms for bug reports, feature requests, documentation, questions,
  and security reports
- Discussion templates for General, Ideas, Q&A, and Show & Tell
- Shared CI workflows: markdown lint, link check, YAML lint, spell check,
  super-linter, stale issue management, Dependabot auto-merge, label sync,
  release drafter, first-interaction greeting, auto-labeling, and CodeQL
  security audit
- Pull request template with review and security checklists
- Organization-wide CODEOWNERS, Dependabot, and editor configuration
