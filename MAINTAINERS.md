# Maintainers

This file lists the current maintainers of HyperSentry Labs and the
organization-wide repositories they are responsible for. Individual
project repositories may list additional, project-specific maintainers in
their own `CODEOWNERS` file.

| Name         | GitHub                                          | Areas                          |
| ------------ | ------------------------------------------------ | ------------------------------- |
| Matin        | [@power0matin](https://github.com/power0matin)   | Organization-wide, all projects |

## Becoming a maintainer

See [GOVERNANCE.md](GOVERNANCE.md#adding-new-maintainers) for the process
by which new maintainers are added.

## Responsibilities

Maintainers listed here are responsible for the files and workflows defined
in this `.github` repository, which are inherited by default across all
public repositories in the organization. Changes to organization-wide
templates, workflows, or policies should be reviewed by a maintainer listed
above.
